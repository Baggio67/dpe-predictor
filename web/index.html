Fondements Mathématiques & Méthodologie

Le modèle : Régression Logistique Multinomiale

La régression logistique multinomiale est une extension de la régression logistique binaire pour les variables cibles ayant plusieurs catégories ordonnées ou non (dans notre cas, les 7 classes énergétiques de A à G).

Pour une observation donnée sous forme de vecteur de caractéristiques \(X\), la probabilité d'appartenir à la classe \(k\) est modélisée par la fonction Softmax :

$$P(Y = k | X) = \frac{e^{\beta_{0, k} + \sum_{j=1}^{M} \beta_{j, k} X_j}}{\sum_{l=1}^{K} e^{\beta_{0, l} + \sum_{j=1}^{M} \beta_{j, l} X_j}}$$

Où :

  • \(K = 7\) représente le nombre de classes énergétiques (A à G).
  • \(M = 14\) est le nombre de variables après prétraitement.
  • \(\beta_{j, k}\) sont les coefficients associés à la variable \(X_j\) pour la classe \(k\).
  • \(\beta_{0, k}\) est l'ordonnée à l'origine (intercept) de la classe \(k\).

Prétraitement & Pipeline

Pour appliquer ce modèle linéaire, nos données subissent les transformations suivantes :

  1. Encodage (One-Hot) : Les variables catégorielles (Type de chauffage et Isolation) sont converties en variables binaires (0 ou 1) pour chaque catégorie.
  2. Normalisation (Standardisation) : Chaque variable \(x_j\) est centrée et réduite sur la base des données d'entraînement :
    $$X_j = \frac{x_j - \mu_j}{\sigma_j}$$
    Où \(\mu_j\) et \(\sigma_j\) sont la moyenne et l'écart-type de la variable \(j\).
